  <tr>
    <td width="50%">
      <h3>🎵 Play Music</h3>

        <li>Instant <b>Search</b></li>

## 🛠️ Build from Source

```bash
git clone https://github.com/Sandeepbedia/AB-Player-v2.git
cd AB-Player-v2
./gradlew installDebug
```

**Built with:** Kotlin · Jetpack Compose · Material 3 · ExoPlayer · Room DB · Hilt
